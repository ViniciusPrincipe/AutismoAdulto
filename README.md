# AutismoAdulto
Previsão de autismo em adultos baseados no dataset Autism Screening on Adults (https://www.kaggle.com/datasets/andrewmvd/autism-screening-on-adults). 
Random Forest para previsão direta. Clusters Kmeans para análise não-supervisionada, além de demonstrar determinismo na construção do dataset, possível sub-notificação.
